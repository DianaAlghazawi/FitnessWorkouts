# Changelog

All notable changes to Offline Fitness Workouts will be documented in this file.

## [1.0.0] - 2024-01-XX

### Added
- Initial release of Offline Fitness Workouts
- 5 workout categories with multiple routines
- Full Body Workouts category
- Abs & Core category
- Arms & Shoulders category
- Legs & Glutes category
- Stretching & Flexibility category
- Workout browsing by category
- Difficulty filtering (Easy, Medium, Hard)
- Search functionality
- Workout details screen with step-by-step instructions
- Built-in workout timer
- Favorites system
- Progress tracking (workouts, minutes, calories)
- Workout history
- Light/Dark theme support
- System theme detection
- Settings screen
- Privacy Policy screen
- Material 3 design
- Accessibility support
- SQLite database for local storage
- Clean Architecture with MVVM pattern
- Riverpod for state management
- Zero permissions required
- 100% offline functionality

### Technical
- Flutter 3.x compatibility
- Dart null-safety
- Clean Architecture implementation
- Repository pattern
- Use case pattern
- Dependency injection with Riverpod
- SQLite database with sqflite
- Material 3 theming
- Responsive layouts

### Compliance

- No third-party tracking
- Privacy policy included
